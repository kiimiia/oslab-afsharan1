
 لینوکس به بزرگ و یا کوچک بودن حروف در
 ترمینال حساس است و خروجی آنها متفاوت میشود

 arch :
نمایش داده می شود x86_64

uptime :    
زمان استفاده از ترمینال 
ساعت جاری
مدت زمان استفاده از لینوکس
تعداد کاربر 
میانگین لود سیستم 

free :
 مقدار رم در دسترس و مقدار های استفاده و آزاد را نشان می دهد 

 pwd :
نام مسیرفعلی را به صورت کامل از روت چاپ میکند  

 exit :
پنجره ی ترمینال را می بندد
